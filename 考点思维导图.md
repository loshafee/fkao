# 法考客观题考点

## 刑法（40题）

### 刑法效力范围
- 空间效力: 以属地原则为基础，兼采属人、保护、普遍原则 → 2020-x01

### 犯罪形态
- 犯罪中止: "能而不欲"，自动放弃 → x01
- 犯罪未遂: "欲而不能"，意志以外原因 → x10
- 犯罪预备: 尚未着手 → x10
- 中止犯可自动放弃 + 有效防止结果 → x04

### 犯罪既遂形态
- 盗窃既遂("失控+控制说")/杀人既遂(结果发生)/爆炸既遂(危险犯)/绑架既遂(行为犯) → 2020-x04

### 正当化事由
- 正当防卫: 正在进行的不法侵害 → x02
- 防卫过当: 应负刑责但应减轻或免除 → x02
- 特殊防卫: 严重危及人身安全的暴力犯罪 → x02
- 紧急避险: 不适用于职务/业务上有特定责任的人 → x15

### 主观心态区分
- 故意杀人vs过失致人死亡: 本质区别在于对死亡结果的主观心态(希望/放任 vs 疏忽/轻信) → 2021-x03

### 认识错误
- 对象不能犯: 意图侵害对象不存在但行为有客观危险性→犯罪未遂 → 2020-x03

### 共同犯罪
- 构成要件: 共同故意 + 共同行为 → x03
- 间接正犯: 利用无刑事责任能力人 → x03
- 同时犯: 缺乏共同故意 → x03

### 单位犯罪
- 单位犯罪双罚制+不以单位犯罪论处的情形(为犯罪设立/以犯罪为主要活动) → 2021-x02

### 因果关系
- 介入因素异常且作用大 → 中断因果 → x07
- 被害人特殊体质不中断因果 → x07
- 不作为犯罪也需要因果关系 → x07

### 刑罚制度
- 累犯: 前后罪均为故意、18岁以上、刑罚执行完毕5年内 → x11
- 特殊累犯: 危害国家安全/恐怖活动/黑社会，不限5年 → x11
- 缓刑: 拘役/3年以下有期，累犯不得缓刑 → x13
- 自首: 投案对象不限(公安/检察/法院/单位) → x08
- 余罪自首(准自首): 被采取强制措施后供述未掌握的其他罪行 → x08
- 追诉时效: 无期/死刑=20年，需最高检核准 → x22
- 刑事责任年龄: 刑修十一→12-14岁个别下调(故意杀人/伤害致死) → x14
- 刑事责任年龄(细化): 刑修十一: 12-14岁犯故意杀人/故意伤害致死+情节恶劣+最高检核准→负刑责 → 2021-x01

### 财产犯罪
- 盗窃罪: 秘密窃取 vs 侵占（合法持有后拒不归还）vs 转化型抢劫 → x05
- 抢劫罪: 携带凶器抢夺=抢劫，入户抢劫加重 → x12
- 抢劫暴力须达压制反抗程度 → x12
- 贪污罪: 国家工作人员利用职务便利+侵吞/窃取/骗取/其他 → x21
- 受贿罪: 国家工作人员+利用职务便利+收受财物+为他人谋利 → x06
- 合同诈骗罪: 合同诈骗罪是诈骗罪的特殊法条，在签订履行合同过程中骗取财物 → 2020-x06

### 人身权利犯罪
- 非法拘禁罪 vs 绑架罪: 是否有勒索财物目的 → x09
- 故意杀人中止: 自动放弃+有效防止结果 → x04

### 妨害传染病防治罪
- 拒绝执行防疫措施+引起甲类传染病传播/传播严重危险 → 2020-x05

### 新型犯罪（刑修十一+）
- 帮信罪: 明知+提供技术支持/推广/支付结算，情节严重 → x16
- 洗钱罪: 7类上游犯罪(毒品/黑社会/恐怖/走私/贪污/金融/金融诈骗) → x17
- 高空抛物罪: 情节犯，无需造成实际损害 → x18
- 危险驾驶罪: 追逐竞驶/醉驾/超载超速/违规运输危化品 → x19
- 袭警罪: 行为犯(暴力袭击)，不要求伤害结果 → x23
- 侵犯公民个人信息罪: 情节犯，主体不限 → x20
- 假冒注册商标罪: 同一种商品+相同商标 → x24
- 催收非法债务罪: 暴力/限制自由/恐吓催收高利贷等非法债务 → x25
- 刑修十一新增罪名: 袭警罪/高空抛物罪/冒名顶替罪/妨害安全驾驶罪/催收非法债务罪等 → 2021-x04
- 网络暴力犯罪: 人肉搜索致人自杀→侮辱罪与侵犯公民个人信息罪想象竞合，择一重罪 → 2022-x03
- 妨害安全驾驶罪想象竞合: 暴力殴打驾驶员致车辆失控→妨害安全驾驶罪与以危险方法危害公共安全罪想象竞合 → 2023-x01
- 自洗钱入罪: 刑修十一将自洗钱入罪，罚金刑改为无限额 → 2023-x02
- 帮信罪vs掩饰隐瞒犯罪所得罪: 概括明知网络犯罪→帮信罪；具体明知是犯罪所得→掩饰隐瞒犯罪所得罪 → 2023-x03
- Deepfake犯罪: 深度伪造淫秽视频传播→诽谤罪与传播淫秽物品罪想象竞合 → 2024-x03

### 反有组织犯罪法
- 有组织犯罪=黑社会性质组织犯罪+恶势力组织实施的犯罪 → 2022-x01

### 刑修十二（2024）
- 行贿罪从宽条件收紧，行受贿并重惩处 → 2024-x01
- 民企内部腐败入罪: 非法经营同类营业/为亲友非法牟利/徇私舞弊低价折股扩展至非国企 → 2024-x02

### 醉驾新规
- 血液酒精<150mg/100ml且无从重情节→可不认为是犯罪 → 2025-x01

### 刑事合规出罪
- 合规整改为酌定从宽情节，第三方监督评估机制考察 → 2025-x02

### AI犯罪归责
- AI是犯罪工具非犯罪主体，利用AI犯罪由行为人承担刑责 → 2025-x03

### 危险作业罪想象竞合
- 拒不执行停产令+发生重大事故→危险作业罪与重大责任事故罪想象竞合 → 2025-x04

## 民法（40题）

### 民法基本原则
- 绿色原则: 民法典六大基本原则之一，民事活动应节约资源保护生态环境 → 2020-m03

### 民事主体
- 限制民事行为能力人: 超出年龄智力的合同→效力待定 → m01

### 物权
- 不动产登记: 所有权转移以登记为生效要件 → m02
- 善意取得(动产): 无权处分+善意+合理价格+交付 → m03
- 善意取得(不动产): 需登记完成 → m03
- 遗失物/盗赃物: 不适用善意取得 → m03
- 拾得遗失物: 应返还权利人，公告1年无人认领→归国家 → m15
- 抵押权vs质权清偿顺序: 按登记/交付先后 → m06
- 留置权: 合法占有+债务到期+同一法律关系 → m20
- 居住权: 合同/遗嘱设立+登记生效+人身专属性+不得转让继承 → m05
- 土地承包经营权: 耕地30年/草地30-50/林地30-70,可流转 → m22
- PMSI价款债权抵押权: 动产抵押担保价款债权，交付后10日内登记→超级优先权(但留置权人除外) → 2021-m01, 2022-m01

### 合同法
- 合同效力: 可撤销(重大误解/欺诈) vs 无效(恶意串通/违法) → m07
- 要约邀请: 商业广告内容符合要约条件视为要约 → m14
- 承诺: 实质性变更→新要约,非实质性变更→有效 → m24
- 格式条款: 须合理提示,不利于提供方的解释,部分无效不影响整体 → m18
- 保证合同: 从合同,一般保证有先诉抗辩权,连带保证无 → m10
- 不可抗力: 不可抗力=不能预见+不能避免+不能克服。免责但有例外(法律另有规定) → 2020-m01
- 情势变更: 合同基础条件发生不可预见的重大变化，继续履行显失公平→可重新协商/请求法院变更解除 → 2020-m02
- 保管合同贵重物品: 寄存贵重物品未声明，毁损灭失后→按一般物品赔偿 → 2020-m04
- 非金钱债务履行例外: 法律/事实不能履行、不适于强制履行/费用过高、债权人合理期限未请求→除外 → 2021-m02
- 重大误解认定: 对行为性质/对方当事人/标的物品种质量规格价格数量等错误认识→可撤销 → 2023-m01
- 可预见性规则: 违约损害赔偿以合同订立时违约方预见或应当预见的损失为限 → 2024-m01
- 交易习惯: 当事人间惯常做法+行业通常做法且对方知道或应当知道→可认定为交易习惯 → 2024-m02
- 可得利益计算: 可得利益=履行后可获利益-履行成本；持续性合同按替代交易合理期限计算；不可与信赖利益同时主张 → 2024-m03

### 侵权责任
- 饲养动物: 饲养人/管理人无过错责任(交付即转移) → m08
- 高空抛物: 具体侵权人负责,难以确定→可能加害人补偿,公安有调查义务 → m23
- 自甘风险: 文体活动,其他参加者无故意/重大过失免责,组织者不免责 → m19

### 婚姻家庭与继承
- 夫妻共同财产: 工资奖金/知识产权收益(婚内取得) → m16
- 个人财产: 人身损害赔偿金/婚前财产 → m16
- 离婚: 宣告失踪+提起离婚→应准予;分居满2年→感情破裂 → m11
- 离婚冷静期: 协议离婚: 30日撤回期+30日领证期，仅适用于协议离婚 → 2020-m05
- 代位继承: 子女先死→晚辈直系血亲代位 → m09/m21
- 遗嘱继承优先于法定继承;公证遗嘱不再优先 → m21

### 债法
- 诉讼时效: 普通3年,届满→债务人取得抗辩权(债权不消灭) → m04
- 时效中断: 催收/部分履行/起诉/同意履行 → m12
- 买卖不破租赁: 所有权变动不影响租赁合同 → m13
- 无因管理: 无约定/法定义务+为他人利益+管理必要费用 → m17
- 不当得利: 无法律原因受有利益致他人受损 → m25
- 不属于不当得利: 明知无债清偿/道德义务给付 → m25

### 责任体系
- 按份责任: 可确定份额按份/难以确定平均 → 2021-m03
- 连带责任: 权利人可向任一/部分/全部主张，超额承担可追偿 → 2021-m03, 2023-m02

### 新公司法相关
- 出资责任: 股东未按期足额缴纳出资→补缴+赔偿公司损失；设立时其他股东承担连带责任 → 2025-m01
- 横向法人人格否认: 股东利用控制的多个公司滥用独立地位和有限责任→各公司对任一公司债务连带 → 2025-m02
- 董监高信义义务: 忠实义务+勤勉义务；谋取公司商业机会须经同意；违反所得→公司归入权；违法违章造成损失→赔偿 → 2025-m03

## 刑事诉讼法（32题）

### 基本原则与制度
- 认罪认罚从宽: 所有案件/所有阶段,一般应当从宽 → ss01
- 认罪认罚"认罪"含义: 自愿如实供述罪行+对指控犯罪事实无异议，对个别情节有异议不影响"认罪" → 2020-ss01
- 法定不追诉(刑诉法16条): 过时效/特赦/死亡/显著轻微 → ss02
- 非法证据排除: 刑讯口供绝对排除,实物证据裁量排除,检方承担证明责任 → ss03
- 值班律师: 非辩护人,可咨询,不能出庭,无阅卷权 → ss12
- 值班律师定位: 值班律师非辩护人，不享有阅卷权和出庭辩护权 → 2021-ss02

### 强制措施
- 拘留: 检/法可决定,公安执行,24小时内通知家属,最长37日 → ss04
- 取保候审: 保证人/保证金择一,公安执行,经批准可离开 → ss08
- 监视居住: 最长6个月,在住处执行,指定居所可折抵刑期 → ss17
- 逮捕: 有证据+徒刑以上+取保不足防危险,检批/法决/公安执行 → ss14

### 辩护与代理
- 委托辩护: 第一次讯问/强制措施之日起,侦查阶段仅律师 → ss05
- 强制辩护: 无期/死刑须有辩护人 → ss05
- 阅卷权: 审查起诉之日起 → ss05
- 辩护律师可独立做无罪辩护(不受认罪认罚限制) → ss01
- 辩护律师会见权: 除危害国家安全/恐怖活动犯罪外，持三证会见→看守所应48小时内安排 → 2024-ss01

### 侦查
- 侦查羁押期限: 逮捕后2个月,可延长至3/5/7个月 → ss15
- 技术侦查: 检/公可实施,需审批,材料可作证据,初次3个月 → ss22

### 审查起诉
- 不起诉种类: 法定(绝对)/酌定(相对)/存疑(证据不足) → ss13/ss10
- 审查批捕: 可讯问,应听取辩护律师意见,逮捕后羁押必要性审查 → ss23
- 提起公诉证明标准: 起诉标准: 犯罪事实清楚、证据确实充分(文字同定罪标准但实践有差异) → 2020-ss02
- 补充侦查: 退回补充侦查以2次为限，每次1个月；检察院可退回公安也可自行侦查 → 2020-ss03
- 合规不起诉: 企业合规整改合格→检察院可不起诉；第三方监督评估机制 → 2024-ss02
- 行刑衔接: 正向(行政→刑事)+反向(刑事→行政)；不起诉但需行政处罚→应提检察意见移送 → 2024-ss03

### 审判程序
- 速裁程序: 不适用于盲聋哑/未成年人/重大影响/无罪辩护 → ss06
- 二审: 上诉不加刑(仅被告方),抗诉可加重,死刑案应开庭,全面审查 → ss07
- 缺席审判: 贪污+危害国家安全等,逃匿境外+传票未到,重审非自动失效 → ss11
- 再审提起主体: 最高法/上级法/本院审判委员会,被害人只能申诉 → ss21
- 证人出庭: 有异议+对定罪量刑有重大影响→应当出庭 → ss18

### 特殊程序
- 刑事和解: 民间纠纷+人身/财产犯罪+3年以下/(过失犯罪除渎职外7年以下) → ss16
- 强制医疗: 暴力+精神病人+危害社会,法院决定,无固定期限 → ss19
- 未成年人: 法定代理人到场,犯罪记录封存,严格限制逮捕 → ss20
- 附带民事诉讼: 可分开审理,可调解,被害人死亡近亲属可提 → ss24

### 证据
- 直接证据: 单独直接证明主要事实(被告人供述) → ss09
- 间接证据: 物证/鉴定意见等不能单独证明 → ss09
- 证据三性: 客观性(客观存在非主观臆断)+关联性(与案件事实客观联系)+合法性(主体程序形式合法) → 2021-ss01

## 民事诉讼法（27题）

### 管辖
- 合同纠纷: 被告住所地+合同履行地 → ms01
- 协议管辖: 书面+实际联系地点+不违反级别/专属管辖 → ms02
- 专属管辖: 不动产/港口作业/继承遗产 → ms09

### 证明
- 证明责任: 谁主张谁举证(客观证明责任不可转移) → ms03
- 高度盖然性: 民事一般标准,确信事实存在具有高度可能性 → ms19
- 自认制度: 一方陈述于己不利事实对方无需举证；特定条件下可撤回(对方同意/受欺诈胁迫重大误解) → 2020-ms01
- 电子数据证据: 微信聊天/电子邮件/网上交易记录/手机短信→均属电子数据 → 2020-ms02
- 举证期限: 一审普通程序≥15日，二审新证据≥10日 → 2020-ms03

### 诉讼程序
- 简易程序: 事实清楚+权利义务明确+争议不大 → ms04
- 小额诉讼: 标的额≤年平均工资50%+金钱给付,一审终审 → ms13
- 在线诉讼: 经当事人同意,与线下同等效力 → ms17
- 在线诉讼(2021修正): 经当事人同意可在线诉讼，与线下诉讼同等效力 → 2021-ms01
- 送达: 直接送达为原则,可电子送达(含判决书),可公告送达(30日) → ms20
- 诉讼代理人: 律师/近亲属/工作人员/团体推荐人 → ms12

### 保全与先予执行
- 财产保全: 诉前/诉中,范围限于请求范围,被申请人提供担保应解除 → ms14
- 证据保全: 法院或公证机关均可 → ms21
- 实现担保物权: 特别程序(非讼),裁定拍卖变卖,可异议 → ms15

### 执行
- 执行异议: 案外人可提,合议庭审查(非执行员) → ms06
- 执行和解: 无强制执行力,恢复原执行/另行起诉,原文书不失效 → ms18

### 特别程序
- 公示催告: 期间≥60日,期满可申请除权判决 → ms16
- 第三人撤销之诉: 6个月内,原审法院,不当然中止执行 → ms05
- 公益诉讼: 污染/消费者权益,检方可提,可调解,撤诉需法院准许 → ms10
